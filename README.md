# JavaScript Quiz App

A simple and responsive quiz application built using HTML, CSS, JavaScript,
and Bootstrap. The app displays multiple-choice questions and calculates
the final score based on user answers.

## Features
- Multiple-choice quiz questions
- Score calculation
- Restart quiz option
- Responsive UI using Bootstrap
- Clean and simple design

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap 5

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any modern web browser.

## Preview
Open the file in your browser and start answering the quiz!

This project is created for learning and practice purposes.
